## 🌟 Featured Projects

### [RLine](https://rline.ryanneeki.xyz) — Full-Stack Social Media Platform
> A production-grade social platform built from scratch with a full frontend/backend split.

Built RLine to understand how real social platforms work under the hood — authentication, feed delivery, user interactions, and all. It handles real users, real data, and is deployed live.

**What I built:**
- JWT-based auth system with secure session management
- RESTful API in Node.js/Express serving a React frontend
- Responsive UI that works across all screen sizes
- Post, like, and user profile features with dynamic content delivery

**Stack:** React · Node.js · Express.js · PostgreSQL · JWT

### [Kowica](https://github.com/Kowica) — Restaurant & Supply Chain Platform *(in progress)*
> QR-based management system for restaurants covering ordering, supply chain, and payments.

Tackling a real-world problem: restaurant operations are fragmented. Kowica ties together table ordering (via QR), inventory tracking, and payment flows into one system.

**Stack:** Node.js · TypeScript · PostgreSQL · Prisma · React
